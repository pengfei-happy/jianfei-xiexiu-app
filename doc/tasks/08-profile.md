# 个人中心与本地数据管理模块任务

目标：实现用户体重设置、计划切换、减脂须知、历史入口和本地数据重置。

## 最小任务清单

- [ ] 建立个人中心页面容器。
- [ ] 实现 `WeightSettingsCard`。
- [ ] 支持填写初始体重。
- [ ] 支持填写目标体重。
- [ ] 体重单位固定为斤。
- [ ] 校验体重输入必须为有效正数。
- [ ] 保存体重设置后更新首页和数据看板进度。
- [ ] 实现 `PlanSwitchCard`。
- [ ] 展示当前计划。
- [ ] 支持切换到秦昊15天计划。
- [ ] 支持切换到大冰7天循环计划。
- [ ] 切换计划时不删除历史记录。
- [ ] 切换计划后影响后续日期食谱和计划日计算。
- [ ] 秦昊15天完成后可从提示进入大冰切换。
- [ ] 实现 `NoticeCard`。
- [ ] 展示每日饮水1800-2000ml。
- [ ] 展示大冰循环每日饮水2000ml。
- [ ] 展示称重标准。
- [ ] 展示无油少盐、无糖、无酱料、无油炸精加工规则。
- [ ] 展示杜绝零食、奶茶、油炸、夜宵、酒精、精加工食品规则。
- [ ] 展示大冰16:8轻断食和进食顺序。
- [ ] 实现嘴馋解馋法则展示入口，内容未确认时标记待确认或等待原文补齐。
- [ ] 实现 `HistoryEntryCard`。
- [ ] 提供历史打卡记录查看入口。
- [ ] 实现 `DangerZoneCard`。
- [ ] 支持重置打卡和体重记录。
- [ ] 支持重置全部本地数据。
- [ ] 实现 `ConfirmResetDialog`。
- [ ] 所有重置操作必须二次确认。
- [ ] 重置后刷新首页、看板、历史列表空状态。

## 验收标准

- [ ] 用户可自行设置初始体重和目标体重。
- [ ] 计划可手动切换且历史数据保留。
- [ ] 减脂须知完整可读。
- [ ] 危险操作必须二次确认。
- [ ] 重置后本地数据状态一致。
