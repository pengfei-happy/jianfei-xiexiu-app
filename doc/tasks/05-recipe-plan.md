# 食谱计划模块任务

目标：实现秦昊15天和大冰7天循环食谱的完整查看、展开详情和饮食打卡入口。

## 最小任务清单

- [ ] 建立食谱计划页面容器。
- [ ] 实现 `PlanSegmentTabs`。
- [ ] 支持秦昊15天计划和大冰7天循环计划切换。
- [ ] 实现 `RecipeDayList`。
- [ ] 秦昊计划展示 Day 1 到 Day 15。
- [ ] 大冰计划展示循环第1天到第7天。
- [ ] 实现 `RecipeAccordionItem`。
- [ ] 一次只允许展开一个天数。
- [ ] 展开状态切换不丢失已保存打卡数据。
- [ ] 实现 `RecipeDetailCard`。
- [ ] 按早餐、午餐、晚餐、加餐顺序展示。
- [ ] 加餐仅在原文提供时展示。
- [ ] 展示阶段名称。
- [ ] 秦昊计划展示当日预计掉秤。
- [ ] 大冰计划展示非数值型预估文案。
- [ ] 展示核心规则。
- [ ] 展示禁忌或注意事项。
- [ ] 长文本自动换行。
- [ ] 实现 `DietCheckInPanel`。
- [ ] 支持选择“按食谱完成”。
- [ ] 支持选择“未按食谱完成”。
- [ ] 支持新增违规进食记录。
- [ ] 支持编辑或删除当天违规进食记录。
- [ ] 保存饮食打卡后更新每日打卡汇总。
- [ ] 未按食谱完成时仍算饮食记录已填写。
- [ ] 未按食谱完成时不计入饮食完成率。
- [ ] 当前日期之外的历史日期支持补打卡。

## 验收标准

- [ ] 秦昊15天餐单完整展示。
- [ ] 大冰7天循环餐单完整展示。
- [ ] 食谱内容不被摘要或改写。
- [ ] 一次只有一个食谱详情展开。
- [ ] 饮食打卡会影响首页和数据看板状态。
