# 公共 UI 组件模块任务

目标：沉淀五大页面共用的卡片、进度条、日期选择、打卡卡片、折线图和底部导航组件。

## 最小任务清单

- [ ] 建立全局主题常量：主色、背景色、警示色、成功色、弱化色。
- [ ] 建立统一间距、圆角、字号常量。
- [ ] 实现 `DataCard` 基础结构。
- [ ] 为 `DataCard` 支持默认、成功、警示、待确认状态。
- [ ] 为 `DataCard` 支持可选点击回调。
- [ ] 实现 `ProgressBar`。
- [ ] 处理 `ProgressBar` 空状态。
- [ ] 处理进度小于0和超过100%的展示。
- [ ] 实现 `DatePicker` 或日期切换组件。
- [ ] 禁止选择未来日期用于打卡。
- [ ] 支持历史日期选择，用于补打卡。
- [ ] 实现 `CheckInCard`。
- [ ] 支持完成、部分完成、未完成、禁用状态。
- [ ] 实现 `LineChart` 封装。
- [ ] 支持最近7天周视图。
- [ ] 支持最近30天月视图。
- [ ] 缺失体重记录日期不补造数据。
- [ ] 同一天多条体重记录取最后一条。
- [ ] 实现 `BottomTabNavigator`。
- [ ] 底部 Tab 包含：首页、食谱计划、数据看板、运动记录、我的。
- [ ] 当前选中 Tab 使用主色展示。
- [ ] 为长食谱文本设置自动换行和滚动安全样式。
- [ ] 为底部 Tab 上方内容预留安全区域。
- [ ] 覆盖首次使用、无记录、待确认、危险操作等通用空状态。

## 验收标准

- [ ] 公共组件可被五大页面复用。
- [ ] 文案在小屏设备不溢出卡片或按钮。
- [ ] 卡片圆角、间距、字号符合详细设计。
- [ ] 警示和危险操作有清晰视觉区分。
- [ ] 折线图在无数据时展示空状态。
